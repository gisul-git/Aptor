🔍 Find Your Extension IDs

This tool helps you find the IDs of all installed browser extensions so you can add them to the detection database.

ℹ️ How to use:
  1. Method 1 (Easiest): Go to edge://extensions, enable Developer mode, and copy IDs visually
  2. Method 2: Use the console script on the extensions page (see Method 2 below)
  3. Method 3: Click "Scan for Extensions" below (only works if API is available)
  4. Add the IDs to frontend/src/lib/extensionDatabase.ts
  5. Test detection in the precheck page

✅ Method 1: Visual Method (RECOMMENDED - No Script Needed)

  1. Go to edge://extensions or chrome://extensions
  2. Enable "Developer mode" (toggle in top-right)
  3. You'll see the ID displayed under each extension
  4. Just copy the IDs you need and share them here!

💡 This is the easiest method - no console script needed!

⚠️ Note: This tool requires the chrome.management API. If you see an error, you may need to run this from a Chrome extension context or use the manual method below.

📋 Method 2: Use Console Script on Extensions Page (RECOMMENDED)

This method works because the chrome.management API is available on the extensions page itself.

  1. Go to edge://extensions or chrome://extensions
  2. Enable "Developer mode" (toggle in top right)
  3. Open Console (Press F12 or Ctrl+Shift+I)
  4. Paste this script and press Enter:
// This works on edge://extensions page
chrome.management.getAll((extensions) => {
  console.log("=== YOUR INSTALLED EXTENSIONS ===\n");
  
  const enabled = extensions.filter(ext => ext.type === "extension" && ext.enabled);
  
  enabled.forEach(ext => {
    console.log(`Name: ${ext.name}`);
    console.log(`ID: ${ext.id}`);
    console.log(`Version: ${ext.version}`);
    console.log("---");
  });
  
  console.log("\n=== CODE TO ADD TO DATABASE ===\n");
  console.log("Copy this code and add to frontend/src/lib/extensionDatabase.ts:\n");
  
  enabled.forEach(ext => {
    const escapedName = ext.name.replace(/'/g, "\\'");
    console.log(`  { id: '${ext.id}', name: '${escapedName}', file: 'icon.png' },`);
  });
  
  console.log("\n=== DONE ===");
  console.log(`Found ${enabled.length} enabled extensions`);
});

Note: This script only works when run on the edge://extensions or chrome://extensions page itself, not on regular web pages. The chrome.management API is only available on special Chrome pages.

📋 Method 3: Manual Copy (If console script doesn't work)

  1. Go to edge://extensions or chrome://extensions
  2. Enable "Developer mode" (toggle in top right)
  3. For each extension, you'll see an "ID" field
  4. Copy the ID and add it to extensionDatabase.ts with format: 
    { id: 'YOUR_EXTENSION_ID', name: 'Extension Name', file: 'icon.png' },
  5. To find the correct file path, click "Details" on the extension and check the manifest.json for web_accessible_resources
