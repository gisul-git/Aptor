## 🌐 Access the Application

**Open your browser and go to:**
```
http://localhost:3002/design
```

## 🎯 Quick Actions

### 1. View Questions
- Click "Question Management" card
- See all design questions
- Try the Publish/Unpublish buttons ✅
- Click Preview to see question details ✅
- Click Edit to modify questions ✅

### 2. Create a Question
- Click "Create Questions" card
- Try AI Generated mode:
  - Select Role: UI Designer
  - Select Difficulty: Beginner
  - Select Task: Landing Page
  - Click "Generate Question"
- Or try Manual Creation mode

### 3. Create an Assessment
- Click "Create New Assessment" card
- Fill in:
  - Title: "My First Design Test"
  - Description: "Test description"
  - Start Time: Select a date/time
  - Duration: 60 minutes
- Enable proctoring if needed
- Select questions
- Click "Create Design Competency Test" ✅

## 📝 Field Explanations

When creating questions manually:

**Constraints** = Rules to follow
- Example: "Use only 3 colors", "Mobile-first design"

**Deliverables** = What to submit
- Example: "High-fidelity mockups", "Prototype link"

**Evaluation Criteria** = How it's judged
- Example: "Visual hierarchy", "Color usage", "Typography"
